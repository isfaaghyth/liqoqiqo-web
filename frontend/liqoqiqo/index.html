Liqoqiqo

Liqoqiqo adalah sebuah aplikasi untuk melakukan mentoring secara kelompok dengan mentor dan lokasi yang berbeda beda.Dimana setiap grup mendapatkan pengetahuan pada mentor masing - masing dan bisa saling mengenal satu sama lain. Dengan adanya aplikasi ini kegiatan antar kelompok akan berjalan lebih struktur kegiatannya dan mendapatkan kajian lebih selain dari pembimbing mentornya masing - masing.
Pada Aplikasi ini beraneka ragam keunikan fitur fitur untuk anda menuntut ilmu ke surga. apa saja ? tentu saja ada :

1. Fitur Login dan logout untuk user masuk dan keluar ke apliasi.
2. Fitur Lokasi(tempat) yang ingin dituju untuk pergi mentoring.
3. Fitur Do'a untuk melihat doa doa harian.
4. Fitur Riwayat untuk user melihat daftar repositori selama mengikuti mentoring.
5. Fitur Mentoring untuk proses kegiatan mentoring berlangsung.
6. Fitur Chat Grup untuk diskusi hasil dari kegiatan mentoring
